API Documentation

  • addUser

    • Creates a user to the database

      In
      {
  "firstName": "string", (2 min, 20 max)
  "lastName": "string", (2 min, 20 max)
  "nickname": "string", (1 min, 20 max)
  "pubU": "string", (base64, 2048 max)
  "userInfosHashCryptPrivU": "string" 
}
    • Out
      {
  "lastInsertedId": "int"
}

  • addSkill

    • Creates a skill into the database

      In
      {
  "idUCreator": "int",
  "mainName": "string", (max size : 20)
  "subName": "string",  (max size: 20)
  "domain": "string", (max size: 15)
  "level": "int", (min value : 0, max value : 8)
  "imgUrl": "string",
  "color": "string"
  "skillInfosHashCryptPrivUC": "string" Infos du créateur hashées et cryptées avec sa clé privée (sha256)
}
    • Out
      {
  "lastInsertedId": "int"
}

  • addCompetence

    • Creates a "competence" into the database

      In
      {
  "idUTeacher": "int",
  "idUStudent": "int",
  "idSkill": "int",
  "revokedDate": "string"
  "masteryLevel": "int" 
  "competenceInfosHashCryptPrivUT" : "string" Infos du teacher hashées puis cryptées avec sa clé privé (sha256) (max size: 2048)

}
    • Out
      {
  "lastInsertedId": "int"
}
